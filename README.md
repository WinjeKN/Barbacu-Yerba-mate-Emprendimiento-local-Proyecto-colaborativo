## Descripción
Sitio web informativo sobre un emprendimiento local que incluye
información sobre su historia, catálogo de productos y contacto.
## Componentes de Bootstrap utilizados
- Navbar.
- Text-center.
- Container.
- Form control(hace que el input ocupe el ancho de la columna).
- Mb-3 (margin bottom sepsra del elemento de abajo.
- Btn estilo y btn primary color.
- W-100 para que el botón tenga el mismo ancho que lo de arriba.
- Row.
- Col y mb-0.
- Flex.
